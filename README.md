# CSharp_FadeAnimation

使用例：

```
Picturebox pic = new Picturebox;
this.Controls.Add(pic);
pic.Fade_out();
```

<p>setOpacity関数は<a href="https://dobon.net/vb/dotnet/graphics/hadeinimage.html">画像を半透明で表示する</a>の一部を使いました。</p>
